
# 💰 Expense Tracker in Python

A simple command-line based expense tracker built using Python.  
This project helps users to record and manage their daily expenses.

---

## 🚀 Features

- ➕ Add Expense
- 📋 View All Expenses
- 💰 Calculate Total Expense
- ❌ Exit Application

---

## 🛠️ Technologies Used

- Python (Basic Concepts)
- File Handling / Lists (depending on your code)
